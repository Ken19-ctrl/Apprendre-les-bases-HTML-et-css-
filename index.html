Apprendre les bases HTML & CSS en 5 jours

Cours HTML

1. HTML signifie HyperText Markup Language. Il sert à structurer une page web.

2. Un document HTML contient les balises <html>, <head>, <body>.

3. <p> représente un paragraphe. <h1> à <h6> sont les titres.

4. <a href=""> crée un lien. <img src=""> affiche une image.

5. <ul> crée une liste à puces, <ol> une liste numérotée.

6. <table>, <tr>, <td> servent aux tableaux HTML.

7. Les formulaires utilisent <form>, <input>, <textarea>.

8. Les attributs comme id, class, src, href définissent les propriétés.

9. HTML5 introduit des balises sémantiques comme <header>, <section>, <footer>.

10. Les commentaires en HTML s’écrivent : <!-- commentaire -->.

Cours CSS

1. CSS signifie Cascading Style Sheets. Il permet de styliser les pages HTML.

2. Il agit sur les éléments HTML via des sélecteurs.

3. Les couleurs s’appliquent avec `color` ou `background-color`.

4. Le texte est géré avec `font-size`, `font-family`, `text-align`.

5. `margin`, `padding`, `border` forment le “box model”.

6. Les positions : `static`, `relative`, `absolute`, `fixed`, `sticky`.

7. Les unités courantes : `px`, `%`, `em`, `rem`.

8. Les coins arrondis : `border-radius`.

9. Les effets : `transition`, `hover`, `opacity`, `animation`.

10. Le responsive design utilise `@media` pour s’adapter aux écrans.
